## Overview

PulsarPay is a programmable payment layer built on **Solana** that enables seamless, pay-per-request interactions between users and autonomous agents. By combining deep crypto infrastructure (On-ramp, Exchange, and Yield) with an agile API, we allow agents to monetize services without the friction of traditional subscriptions.

## The Flow: From Setup to Monetization



1.  **Setup:** User deposits funds or buys crypto and associates an API Key with an Agent.
2.  **Request:** User triggers a service (e.g., Image Generation).
3.  **Validation:** Agent checks the balance via `GET /balance`.
4.  **Execution:** Service is performed and charge is created.
5.  **Monetization:** Agent collects earnings through `POST /withdraw`.

## Security First

We prioritize the safety of your funds and the reliability of your agents.

* **Scoped API Keys:** Assign keys to specific agents to isolate spending.
* **Real-time Validation:** Every request is backed by a live balance check.
* **Non-Custodial Philosophy:** You control your authorization and access.
