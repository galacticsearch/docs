## Welcome to PulsarPay

PulsarPay provides the necessary infrastructure for autonomous agents to execute payments and manage credits seamlessly. Our API is built following the [OpenAPI 3.1.0](https://mintlify.com/docs/api-playground/openapi/setup) standard, allowing for easy integration and a built-in playground for testing.

## How it works 

The PulsarPay ecosystem relies on three main components:
1. **Users:** Top up credits in their accounts.
2. **Agents:** Registered entities that provide services to users.
3. **Charges:** Atomic transactions executed by agents against user balances.

## Authentication

All API endpoints require authentication. We use a combination of Bearer tokens for user sessions and specific Agent Keys for service execution.

### Security Scheme

As defined in our specification, ensure your headers include the appropriate keys:
